CRUD

Para cadastrar funcionario use o parâmetro acao=funcionario
Para cadastrar venda use o parâmetro acao=venda
Para cadastrar cliente use o parâmetro acao=cliente


Para listar funcionarios use o parâmetro acao=listar-funcionarios
Para listar vendas use o parâmetro acao=listar-vendas
Para listar clientes use o parâmetro acao=listar-clientes
